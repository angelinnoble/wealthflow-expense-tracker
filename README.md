# 💰 WealthFlow — Expense Tracker

WealthFlow is a modern and responsive Expense Tracker web application built using HTML, CSS, JavaScript, and Chart.js. It helps users manage income, expenses, budgeting, and financial analytics with interactive charts and a premium dashboard UI.

---

## 🚀 Features

- Add Income & Expenses
- Financial Dashboard
- Real-Time Balance Calculation
- Expense Categories
- Interactive Charts using Chart.js
- Local Storage Support
- Responsive Design
- Modern Premium UI
- Transaction History
- Expense Analytics

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js

---

## 📊 Functionalities

### ✅ Income Management
Users can:
- Add salary
- Freelance income
- Investments
- Bonuses

### ✅ Expense Management
Track:
- Food
- Shopping
- Bills
- Travel
- Entertainment
- Healthcare

### ✅ Analytics
- Income vs Expense Bar Chart
- Expense Breakdown Doughnut Chart
